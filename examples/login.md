Use the Brand-Led UI Designer perspective.

Project:
A login page for a modern finance / crypto product.

Goal:
Help users sign in quickly while making the product feel trustworthy, calm, premium, and well-designed.

Platform:
Responsive web

This should feel:
- calm
- premium
- sharp
- minimal
- trustworthy

Closer to:
- Robinhood restraint
- Apple-like calm spacing
- Linear precision

Avoid:
- generic exchange login page
- loud gradients or glowing backgrounds
- futuristic tech styling
- too many cards / containers
- startup template feel
- over-friendly consumer app softness

What must stand out:
- the sign-in action
- trustworthiness
- clean hierarchy
- brand confidence

What should stay quiet:
- supporting copy
- secondary actions
- legal / footer details
- optional helper text

Desired density:
- sparse to balanced

Constraints:
- Must support email / password login
- Include SSO options
- Include forgot password and sign up entry
- Must work well on mobile and desktop
- Accessibility should meet WCAG AA

What I need from you:
Give me one strong UI direction for this login page, including:
1. Visual intent
2. Why it feels premium
3. Hierarchy strategy
4. Brand expression beyond color
5. UI foundations (type, spacing, surfaces, radius, border, elevation)
6. Component recommendations
7. What to remove or simplify
8. Anti-generic review

Do not optimize for completeness first. Optimize for visual judgment first.
