## 中文 — 长一点版本

最近把一个想法整理成了公开 repo：

**Brand Interface Designer**  
<https://github.com/yoooooowe/brand-interface-designer>

它是一个给 UI 设计 agent 用的 prompt kit。  
核心不是让 AI 更会生成“完整规范”，而是让它更会做**审美判断**。

因为我越来越觉得，很多设计类 prompt 的问题不是“不够专业”，而是太容易掉进一种默认的 SaaS 审美：
- 结构没问题
- 组件也完整
- accessibility / handoff 也能讲
- 但最后就是不够高级，也不够像一个真正有判断的设计师

所以这套东西主要在补这些能力：
- hierarchy
- typography
- spacing / restraint
- 品牌感不只靠颜色
- anti-generic critique
- 做减法，而不是一直加东西

里面现在有：
- 主 agent prompt
- taste reference
- critique mode
- 常见页面 examples
- 中英文 README

如果你也在折腾 design agent，或者也烦 AI 一张嘴就是“clean modern SaaS UI”，可以看看。
